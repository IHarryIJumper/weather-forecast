# Weather Forecast

## Local project launch

### Installing

Install [Webpack](http://webpack.github.io):

```
npm install webpack -g
```

Install project packages:

```
npm install
```

### Building

Development build:

```
npm run dev
```

Production build:

```
npm run build
```


### Start project

```
npm start
```

### Project route

```
http://localhost:3000/
```
